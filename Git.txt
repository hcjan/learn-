Git(二进制文件只能看到增加减少文件 无法看到真实改动)
Pwd 显示当前目录
Mkdir 创建新目录
Git init 变为git可管理	的仓库
Ls -ah 查看目录先文件
Cd .. 返回上一级目录
Git add 把文件加入到缓存区
Git commit -m”  ” m后面写改动信息
git status 查看仓库
git diff 查看文件更改不同（暂存区和工作区）
git log 查看提交记录
cat 查看文件内容 
git reset --hard HEAD^ 版本回退（回退后想反悔找到没关的log 找到版本号还原eg：git reset --hard 1215a）
git reflog 记录每一次的更改
git diff HEAD -- Git.txt 查看工作区和版本库的不同
